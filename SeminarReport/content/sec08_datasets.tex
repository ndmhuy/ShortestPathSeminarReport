\subsection{Header Format}
Each dataset is emitted with metadata header lines, where the dataset name is taken from the output filename:
\begin{verbatim}
# Directed <dataset_name> graph dataset
# Nodes: <V> Edges: <E>
# FromNodeId    ToNodeId
\end{verbatim}

followed by weighted edges in the form:
\begin{verbatim}
from to weight
\end{verbatim}

\subsection{Benchmark Interpretation}
The DAG dataset is important because it highlights two different costs:
\begin{itemize}
	\item \texttt{dagShortestPath}: recomputes the topological traversal each run.
	\item \texttt{dagShortestPath (precomputed topo order)}: reuses one topological order across source runs and is the fairer textbook-style comparison.
\end{itemize}
